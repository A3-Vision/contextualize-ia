# Glossário em linguagem clara

Referência **opcional** (pode ir para dentro do `CONTEXT.md` do usuário). O núcleo do pacote continua sendo o **[PRD base](../templates/PRD-BASE-AGENTE-IDE.md)** + **[TEMPLATE-CONTEXT](../templates/TEMPLATE-CONTEXT.md)**.

Uma página para **qualquer pessoa** usar no próprio `CONTEXT.md` ou imprimir junto do checklist — sem jargão desnecessário.

| Palavra | Significado simples |
|---------|---------------------|
| **Repositório** | Sua “pastinha de projeto” na nuvem + cópia no computador — tudo versionado junto. |
| **Git** | Sistema que **guarda histórico** das mudanças nos arquivos (quem mudou o quê e quando). |
| **Commit** | “Tirar uma foto” do estado dos arquivos **neste momento** — com uma mensagem curta (ex.: “atualizei regras”). |
| **Push** | Enviar essas mudanças para a **nuvem** (GitHub). |
| **Clone** | Baixar pela primeira vez o repositório da nuvem para uma pasta no PC. |
| **Markdown (.md)** | Arquivo de texto com **títulos** (`#`) e listas — parece anotação; abre em qualquer editor. |
| **CONTEXT** | Arquivo **manual** para você e para a IA: regras, mapa de pastas, o que não fazer. |
| **PRD** | Texto que explica **por que** o repositório existe e o que entra / não entra (1 página). |
| **Agente de IA** | Chat que ajuda — **se** você der o manual (`CONTEXT`), ele erra menos. |
| **IDE** | Programa “forte” para quem programa — **não** é obrigatório na **Trilha guiada**. |
