Como usar no seu editor

Cole o PRD no agente da IDE; ele monta a estrutura em contexto/ e um CONTEXT.md vivo com você.

Estas instruções cobrem só o fluxo IDE + PRD. Para servidor HTTP na raiz (navegador, explorador, rede), veja Boas-vindas → Ver o pacote no navegador.

O pacote entrega uma base: um PRD em Markdown feito para você colar no chat do Cursor, VS Code (Copilot), Antigravity ou outro agente que leia a pasta do projeto.

O agente usa esse texto para criar a estrutura dentro de contexto/ na raiz do repositório (pastas, contexto/CONTEXT.md, README, docs/, decisoes/ quando fizer sentido). O manual não mistura com o pacote em cliente/. O CONTEXT.md deve ser vivo: o agente faz perguntas, você responde, e o manual é atualizado junto com o seu contexto — nada de “template morto”.

Passos

  1. Crie uma pasta vazia (ou abra um repositório Git) no seu computador.
  2. Abra essa pasta no Cursor / VS Code / Antigravity.
  3. Abra o arquivo cliente/templates/PRD-BASE-AGENTE-IDE.md deste pacote, selecione tudo e cole no chat do agente (ou anexe o arquivo, se a IDE permitir).
  4. Responda às perguntas do agente. Deixe que ele proponha a árvore sob contexto/ e o primeiro contexto/CONTEXT.md — depois vocês refinam juntos.
Onde está o arquivo: na pasta do pacote, em cliente/templates/PRD-BASE-AGENTE-IDE.md. O template completo de CONTEXT.md está em cliente/templates/TEMPLATE-CONTEXT.md — o agente pode mesclar com o que você conversar.

Material extra (opcional)

Guia longo em Markdown, glossário e FAQ estão em cliente/conteudo/ se quiser aprofundar fora do fluxo “só colar o PRD”.
