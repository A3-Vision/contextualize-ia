Contextualize IA

Explorador — pasta contexto/

Árvore gerada a partir de estrutura-manifest.json. Arquivos .md são renderizados aqui. Com servidor HTTP na raiz do pacote (ex.: npx serve) a pré-visualização Markdown é completa. Abrir por file:// lista a árvore (manifesto em estrutura-manifest.js), mas o navegador pode bloquear a leitura dos .md; nesse caso use o link “Abrir em novo separador” ou suba o servidor. Comandos do servidor: Boas-vindas → Ver o pacote no navegador.
