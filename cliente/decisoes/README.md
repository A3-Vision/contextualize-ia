# Decisões

Um arquivo por decisão importante. Nome sugerido: `ADR-NNN-titulo-curto.md` (ver template em [../templates/TEMPLATE-ADR-001-exemplo.md](../templates/TEMPLATE-ADR-001-exemplo.md)). No começo pode ser só `minha-decisao.md` com data e texto — sem formalidade.

Esta pasta no **produto** guarda ADRs sobre **o próprio pacote** (ex.: mudança de escopo, formato de distribuição). Quem usa o método cria a pasta `decisoes/` no **próprio** repositório de contexto.
